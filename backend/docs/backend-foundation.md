# Backend Foundation

This document describes Prompt 1A scope for Developer 2 on branch `feature/Product-Seller`.

## Included

- FastAPI application factory
- Pydantic Settings configuration
- Async SQLAlchemy foundation
- Alembic async migration setup
- Common response, error, and pagination models
- Dependency injection helpers
- Structured logging
- Request, performance, security, CORS, trusted host, and GZip middleware
- Centralized exception handlers
- Health, liveness, and readiness APIs
- Security placeholders for future auth integration
- Async pytest foundation
- Docker and Docker Compose configuration
- Ruff, Black, isort, mypy, EditorConfig, and pre-commit configuration

## Excluded

- Authentication implementation
- Seller, product, category, brand, inventory, warehouse, dashboard modules
- Customer, cart, wishlist, checkout, order, payment, review, notification, AI, analytics, and admin modules
- Business tables, CRUD APIs, dummy data, fake services, and mock APIs

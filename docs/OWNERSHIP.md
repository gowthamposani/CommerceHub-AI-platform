# Ownership

## Developer 3

Developer 3 owns:

- Admin module
- Analytics surfaces
- AI Product Description Generator
- Notification framework
- Admin frontend
- Backend testing infrastructure
- Playwright testing infrastructure
- Docker setup
- Developer 3 documentation

Developer 3 must not implement:

- Authentication
- Customer module
- Seller module
- Product module
- Orders module
- Inventory module
